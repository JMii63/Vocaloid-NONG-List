# Vocaloid NONG List
a bunch of Vocaloid NONGs for use in Geometry Dash (specifically Jukebox)

This is mainly used in my upcoming GDPS "Project Dash: Geometrical Stage", but you can use it in your own levels on the main servers if you want.

I might split some of this into a seperate list (between Vocaloid and SEKAI versions, etc.) but for now it's all in one amalgamation

# How To Use

1. Go to Jukebox settings
2. click on the View button next to "Indexes"
3. Add the .json (link is https://raw.githubusercontent.com/JMii63/Vocaloid-NONG-List/main/in-game.json)
4. Go to a level using any of these NONGs (currently only World Is Mine is available)
5. Use the in-game version from Vocaloid NONG List!

If you want to add a song, please make a comment on the megathread issue
